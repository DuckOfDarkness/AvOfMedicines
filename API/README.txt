Projekt zrealizowany za pomocą ORM sequelize przy użyciu bazy MySQL2.
Projekt posiada zaimplementowane, przykładowe dane biznesowe.

MySQL2 (2.3.3):

name db: tin_sequelize
host: localhost
port: 3306
user: root
passwd: root

Uruchomienie: 
- instalacja pakietów mysql2 i sequelize
- w konsoli, w glownym folderze projektu:  
	* npm i
	* npm start

Dostęp do witryny:

http://localhost:3000/
